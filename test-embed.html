Available Parameters

  • data-key - Your PostHog API key (use "demo" for testing)
  • data-style - Counter style: retro, digital, minimal, classic, neon
  • data-days - Timeframe in days or "all" for all time
  • data-label - Custom label text
  • data-dark - Dark mode: auto, light, dark
